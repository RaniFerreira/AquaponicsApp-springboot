## 📚 Sobre o Repositório

Este repositório foi criado com o objetivo de **praticar o desenvolvimento utilizando Spring Boot e fortalecer as habilidades de programação**.



## 🎯 Objetivos

- Praticar desenvolvimento back-end com Spring Boot  
- Construir e entender APIs REST  
- Reforçar conceitos fundamentais da programação  
- Aprender e aplicar arquitetura MVC  
- Melhorar a lógica e a resolução de problemas  
- Adquirir experiência com estrutura de aplicações reais  

## 🧠 Motivação

A prática constante é essencial para evoluir como desenvolvedor(a).  
Este repositório serve como um espaço para **praticar, testar e acompanhar a evolução no aprendizado de Spring Boot**.
